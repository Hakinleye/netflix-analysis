# Netflix Data Analysis

## Overview
This project analyzes Netflix's dataset to gain insights into its content distribution, trends over the years, and key statistics on movies and TV shows.

## Dataset
	•	Source: Netflix Titles Dataset (Kaggle)
	•	Columns: type, title, director, cast, country, date_added, release_year, rating, duration, listed_in (genres)

## Technologies Used
	•	Python (Pandas, NumPy)
	•	Data Visualization (Matplotlib, Seaborn)
	•	Jupyter Notebook

## Steps Performed
1. *Data Cleaning*: Converted date formats, handled missing values, and filtered relevant columns.
2. *Exploratory Data Analysis (EDA)*: Analyzed the distribution of movies vs. TV shows, content release trends, and genre popularity.
3. *Visualization*: Created histograms, bar charts, and line plots to visualize trends in content addition.
4. *Key Insights Extraction*: Derived meaningful conclusions from the dataset.

## Key Insights
- Netflix has seen a significant increase in content over the past decade.
- Movies make up a larger portion of Netflix's catalog compared to TV shows.
- Yearly Content Growth indicates Netflix’s content grew rapidly after 2015.
- The most common release year for content is recent years, indicating a strong focus on fresh content.
- Popular genres include Drama, Comedy, and Action.
- The USA, India, and UK produce the most Netflix content, they are top Content-Producing Countries.
